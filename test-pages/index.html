PayClearly Extension Test Pages

Testing Instructions

  1. Open the extension popup and click "Sign In" (uses test data)
  2. Click "Get Next Payment" to fetch a test payment
  3. Navigate to the test pages below to test different flows:
    • Login: Test portal detection on login page
    • Payment: Test form detection, autofill, and learning mode
    • Confirmation: Test confirmation detection and evidence capture
  4. Check the extension popup for notifications and state changes
  5. Check browser console (background service worker) for logs
